# Lobo Portfolio 
[![NPM](https://img.shields.io/npm/l/react)](https://github.com/brunocesar2/lobo-portfolio/blob/main/LICENSE)

# Sobre o projeto

https://lobo-portfolio.brunocesar.vercel.app

Lobo Portfolio é um projeto criado no curso de HTML/CSS da Origamid.

O projeto consiste em um portfólio fictício para aplicar os conceitos de Grid Layout e Media Queries adquiridos durante o curso.
